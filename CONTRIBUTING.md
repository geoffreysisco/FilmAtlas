# Contributing to FilmAtlas

Thank you for your interest in contributing.

## Reporting Bugs

Before opening a bug report:

1. Search existing issues to avoid duplicates.
2. Use the **Bug Report** template.
3. Include:
   - clear reproduction steps
   - screenshots or screen recordings if possible
   - device and Android version
   - relevant logcat output

## Development Setup

1. Clone the repository.
2. Open the project in Android Studio.
3. Add your TMDB API key to `local.properties`:

   TMDB_API_KEY=YOUR_API_KEY
4. Sync Gradle and run the project.

## Pull Requests

Pull requests should:

- address a clearly defined issue
- keep changes focused and minimal
- follow the existing code style
- avoid unrelated refactors

Small, surgical fixes are preferred.

## Code Style

- Java
- MVVM architecture
- Follow existing naming and formatting patterns

If you're unsure about a change, open an issue first so we can discuss the approach.
